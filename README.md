# BEA - Remove prefix

Allow to remove prefix when in multisite and/or composer.

It will remove the prefixes added by multisite installs and also handle the custom folder for WordPress when in Composer installation.

# Compatibily

Now compatible with :
* single site
* multisites
* multinetworks
* "composer"
